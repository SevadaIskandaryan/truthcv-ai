# TruthCV AI

**Analyze candidate credibility using resume and GitHub signals**

---

## 🚀 Overview

**TruthCV AI** is an AI-powered tool that helps recruiters quickly evaluate candidates by analyzing resumes and cross-checking them with GitHub activity.

Instead of manually reviewing profiles, TruthCV generates **clear, structured insights** about a candidate’s experience, activity, and potential inconsistencies.

---

## 🎯 Why this project?

Recruiters often spend hours reviewing resumes with limited visibility into real work.

TruthCV AI helps answer:

* Does the candidate’s experience match their actual work?
* How active are they on GitHub?
* What should I ask in the interview?

---

## 🧠 Features

* 📄 **Resume Analysis**

  * Extracts skills, experience level, and key signals

* 🧑‍💻 **GitHub Integration**

  * Fetches public repositories and activity
  * Identifies top languages and recent work

* ⚖️ **Credibility Signals**

  * Highlights potential mismatches between resume and GitHub
  * Generates structured insights (not binary judgments)

* 💬 **Interview Question Generator**

  * Suggests tailored questions based on candidate activity

* 📊 **Candidate Summary**

  * Clear, recruiter-friendly report in seconds

## 🏗️ Architecture

```
React Frontend → Django REST API → GitHub API + Resume Parser → Analysis Engine → Response
```

---

## ⚙️ Tech Stack

* **Frontend:** React (TypeScript)
* **Backend:** Django + Django REST Framework
* **APIs:** GitHub REST API
* **AI / Processing:** Prompt-based analysis & rule-based scoring
* **Deployment:** Vercel (frontend), Render/Railway (backend)

## ⚠️ Disclaimer

TruthCV AI does **not** make definitive judgments about candidates.

It provides **signals and insights** to assist recruiters, not replace decision-making.
